# 维修记录统计报告说明

## 概述

从设备出勤统计表的单元格批注中提取维修记录，经台账匹配和故障分类后，生成包含 10 个统计 sheet 的 Excel 报告。

- **输入**：设备出勤统计表（单文件或文件夹）
- **输出**：`维修记录统计.xlsx`（10 个 sheet）
- **排序**：全表按日期 → 标准设备名称排序

---

## Sheet 1：维修明细

全量记录（含非故障），每条维修事件一行。

| 列 | 类型 | 说明 |
|----|------|------|
| 日期 | YYYY-MM-DD | 排序主键 |
| 原始设备名称 | 文本 | 出勤表中的原始车号 |
| 标准设备名称 | 文本 | 台账匹配后的标准名，排序次键 |
| 设备型号 | 文本 | 从标准名提取（去除 `XX#NNNN` 后缀） |
| 原因 | 文本 | 检修 / 点检 / 保养 |
| 班次 | 文本 | 白班 / 夜班 / 未标注 |
| 大类 | 文本 | 14 大类之一，非故障记录为空 |
| 小类 | 文本 | ~60 小类之一，非故障记录为空 |
| 是否故障 | 文本 | 是 / 否 |
| 维修内容 | 文本 | 批注原文，无批注则为空 |
| 工时_分钟 | 数值 | 单元格中的维修工时 |

## 输出模式

### 默认模式

生成单个文件：`维修记录统计.xlsx`，包含上述 10 个 sheet。

### 按年份拆分模式

勾选「按年份拆分输出」后，每年生成独立文件，另附汇总文件：

```
维修记录统计_2023年.xlsx
维修记录统计_2024年.xlsx
维修记录统计_2025年.xlsx
维修记录统计_2026年.xlsx
维修记录统计_汇总.xlsx      ← 包含全部年份数据
```

每个年度文件和汇总文件均包含完整的 10 个 sheet。

---

## Excel 输出格式

- 表头：加粗、蓝底白字（#4472C4）、居中对齐、自动换行
- 数据行：交替底色（白色 / #F2F7FB）
- 列宽：自适应内容（支持中日韩宽字符），最小 8，最大 50
- 日期列：`yyyy-mm-dd` 格式
- 百分比列：`0.00%` 格式
- 小时列：`0.0` 格式
- 冻结首行
- 自动筛选

---

## 故障率计算

```
故障率 = 总故障分钟 ÷ (统计跨度天数 × 24 × 60)
```

- **统计跨度天数** = 设备首次出现日期到最后出现日期的天数 + 1
- 基于全量记录（含非故障）计算跨度，而非仅故障记录
- 结果为小数，Excel 中显示为百分比格式

## 分类规则

默认 14 大类，约 60 小类，通过关键词子串匹配，按行顺序先命中先分类。

| 大类 | 小类数 |
|------|--------|
| 发动机 | 15 |
| 变速箱 | 5 |
| 液压系统 | 5 |
| 电气系统 | 10 |
| 制动系统 | 5 |
| 转向系统 | 4 |
| 悬挂/车架 | 6 |
| 轮胎/轮马达 | 5 |
| 润滑系统 | 4 |
| 空调 | 3 |
| 排气/尾气处理 | 3 |
| 事故/损失 | 3 |
| 日常维护 | 6 |
| 其他 | 1（未分类兜底） |

可通过 Excel 配置模板自定义（管理 → 维修分类配置 → 导出/导入）。
